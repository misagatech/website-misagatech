# MisaGatech - Website

Landing page profesional para MisaGatech Digital Solutions.

## Descripción

Sitio web corporativo para MisaGatech, una empresa especializada en diseño y desarrollo de soluciones digitales. El sitio presenta servicios de diseño web, UX/UI, redes sociales y branding.

## Características

- Diseño responsivo y moderno
- Animaciones sutiles y atractivas
- Formulario de contacto funcional
- Secciones claramente definidas:
  - Hero con llamada a la acción
  - Servicios ofrecidos
  - Sección "Nosotros"
  - Formulario de contacto
  - Footer completo
